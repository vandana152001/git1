Git commands of day1 class

  1. git -v =it is used to check the version of the git
  2. git init=it is used to intialize the git
  3. git add filename=it is used to add a single file
  4. git add .=it is used to add all the files
  5. git commit -m=give any message
  6. git remote add origin link=to connect the github and git repositories
  7. git branch -M main= to create branch
  8. git push origin main
